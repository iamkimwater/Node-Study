	- '에스큐엘' 혹은 '쿼리' 라고 부름

* Sequelize
	- DB 작업을 쉽게 할 수 있도록 도와주는 라이브러리
	- ORM (Object-relational Mapping)
	- 자바스크립트 객체와 관계형 DB를 연결
